## A sneaker API for eCommerce applications
In progress

## Description : 
    it's a free eCommerce API for fetching sneakers

### Requests: 
    Allowed-method : GET

### Actions : 
    Request sneakers : 
        • all available sneakers
        • by brand's name
        • By gender men/women (larger categories will be soon available)
        • By max price
        • by id (uuid)
